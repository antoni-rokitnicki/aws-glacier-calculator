Assumptions

  1. 1 month equals 30 days
  2. Request costs are assumed to be zero
  3. Data transfer rate beyond 350TB are assumed to be the the same as the 350TB tier
  4. You can retrieve up to 5% of your stored data for free each month
  5. You are downloading at a constant rate

Welcome to the unofficial AWS Glacier Calculator.

This is a single page javascript app that aims to help you estimate the cost of AWS glacier. It is not endorsed by Amazon AWS. The rates are referenced from the Glacier pricing page as well as the Glacier FAQ. This page is built with AngluarJS and hosted on Github pages.
